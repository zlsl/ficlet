Внимание!

Для работы поиска по фэндомам необходимо загрузить каталог фэндомов нужной категории, если он еще не был загружен. Для этого с главного экрана в вкладке "КАТЕГОРИИ" необходимо один раз зайти в нужный раздел. Загрузка может занять до одной минуты! Значек рядом с категорией означает, что данные еще не загружены.

Как работает кэш

При первой загрузке ленты она помещается в файловый кэш и в следующий раз загрузка будет произведена из кэша. Для обновления ленты - жест "потянуть вниз".

Кэш можно очистить вручную через меню на главном экране.

Читалка

При первом открытии книги происходит разбивка на страницы. Для крупных фанфиков это может занять заметное время. Последующие загрузки страниц производятся из кэша.

Листать страницы можно свайпом влево-вправо, тапом по экрану, кнопками громкости.

Для регулироки яркости экрана - свайп по экрану в вертикальном направлении. Вверх - увеличить яркость, вниз - уменьшить.
